# PHP-To-Do-List!
Login, Logout, Signup işlemleri ve profil sayfası bulunan bir To Do List. Listeden task silebilir, ekleyebilir, filtreleyebilir ve editleyebilirsiniz. Anasayfada tüm kullanıcıların task'ları listelenirken profil sayfasında sadece o kullanıcının task'ları bulunur.
